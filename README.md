# DoraColor

## Usage
```.swift
import UIKit
import DoraColor

class ViewController: UIViewController {
    
    override func viewDidLoad() {
        super.viewDidLoad()
        view.backgroundColor = .doraColor
    }
}
```

## Installation
### CocoaPods
- Add pod 'DoraColor' to your Podfile.
- Run pod update.
